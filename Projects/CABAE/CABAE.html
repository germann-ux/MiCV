CABAE: Cifrado Cuántico Adaptativo

Descripción del Proyecto

CABAE es un sistema de cifrado de nueva generación que combina principios de la computación cuántica, como la aleatoriedad cuántica y el entrelazamiento, con técnicas clásicas de seguridad. El objetivo es desarrollar un modelo de cifrado híbrido, adaptable y altamente resistente para entornos donde la seguridad de la información es crítica.

El proyecto está actualmente en fase de desarrollo conceptual y experimental. Se explora el uso de Qiskit y Azure Quantum para simular qubits y aprovechar sus propiedades en la generación de claves aleatorias e intransferibles. CABAE también contempla la implementación de heurísticas de detección, telemetría y una interfaz visual moderna para su eventual interacción.

“CABAE no es solo cifrado: es un intento por anticiparse al futuro de la seguridad digital desde hoy.”

Galería del Proyecto

Tecnologías Utilizadas

  • Qiskit (IBM) para simulación cuántica
  • Azure Quantum para pruebas y experimentación
  • Python como lenguaje principal
  • Algoritmos adaptativos de cifrado (híbrido clásico-cuántico)
  • HTML/CSS para interfaz visual del proyecto (todavia sin definir)
  • Simulador local en desarrollo
  • Jupyter Notebooks para documentación y pruebas

Próximos Pasos

  • Probar el prototipo de CABAE en un simulador cuántico real
  • Establecer el modelo formal de generación de claves usando aleatoriedad cuántica
  • Desarrollar y probar la heurística de cifrado híbrido con evaluación comparativa
  • Diseñar una interfaz visual para pruebas y demostraciones
  • Incluir un módulo de telemetría para auditoría interna y fallas
  • Preparar una publicación técnica con la estructura matemática y conceptual
