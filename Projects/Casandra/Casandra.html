Casandra: IA de Análisis Criminal

Repositorio Wiki técnica

Descripción del Proyecto

Casandra es una inteligencia artificial desarrollada para el análisis criminal a partir de datos reales del estado de Guanajuato. Su objetivo es detectar patrones delictivos, generar visualizaciones comprensibles y redactar reportes ejecutivos de forma autónoma con ayuda de Gemini 2.5.

Funciona bajo una arquitectura modular en Python, con visualizaciones automáticas y un motor textual inteligente, transformando datos crudos en conocimiento para la toma de decisiones estratégicas en seguridad pública.

“Casandra es el puente entre el dato y la acción, entre el crimen y su comprensión.”

Galería del Proyecto

Tecnologías Utilizadas

  • Python 3.11
  • Pandas, Numpy, Scikit-learn
  • Matplotlib, Seaborn, Plotly
  • Gemini 2.5 Pro vía API REST
  • Arquitectura modular desacoplada
  • CLI interactiva con `main.py`
  • Generación de HTML programática (`html_builder.py`)
  • Dashboards y visualizaciones exportables
