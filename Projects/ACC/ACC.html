ACC: Aprendiendo C# con Charp

Repositorio Wiki técnica

Descripción del Proyecto

ACC (Aprendiendo C# con Charp) es una plataforma educativa para aprender programación en C# desde cero hasta un nivel avanzado, todo guiado por una inteligencia artificial llamada Charp, tu maestro digital.

El proyecto combina teoría, prácticas interactivas, juegos didácticos (como crucigramas y memoramas), evaluaciones automáticas, un compilador de c# en tiempo real y seguimiento de progreso personalizado. ACC está diseñado para ser modular, expansible y adaptable a distintos niveles de conocimiento, desde principiantes hasta programadores intermedios.

(Originalmente fue pensado para una competencia, ahora quiero darle otro rumbo)

Galería del Proyecto

Tecnologías Utilizadas

  • C# 12 con .NET 8
  • Blazor WebAssembly y MAUI Blazor
  • SQL Server 2022 + Entity Framework Core
  • Roslyn (.NET Compiler Platform) para compilación dinámica
  • Arquitectura distribuida con .NET Aspire
  • Comunicación entre servicios vía DTOs y capas desacopladas
  • Redis para caché distribuido y rendimiento
  • AutoMapper, FluentValidation y Serilog
  • Contenerización con Docker
  • Telemetría y monitoreo con OpenTelemetry
  • Pruebas unitarias con xUnit y Moq
  • Chatbase para IA Charp (orientada a enseñanza de C#)
  • Git, GitHub y Postman para control y pruebas
